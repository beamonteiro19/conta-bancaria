# Conta Bancária

Este projeto simula operações básicas de uma conta bancária utilizando **TypeScript**.

## Funcionalidades
- Criar conta
- Depositar valores
- Sacar valores
- Consultar saldo

## Como rodar
```bash
git clone https://github.com/beamonteiro19/conta-bancaria.git
cd conta-bancaria
npm install
npm run start
